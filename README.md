## Details
This is Simple BMI Calculator

The BMI is a convenient rule of thumb used to broadly categorize a person as
1) Underweight
2) Normal weight
3) Overweight

Step :
- Select Gender
- Choose Height
- Set Weight
- Set Age
- Click on Calculate
